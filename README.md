# Brain Games


Коллекция из 5 мини‑игр на проверку различных математических навыков:

* **Проверка на чётность** — определите, чётное число или нет.
* **Калькулятор** — вычислите результат выражения (+, −, *).
* **Наибольший общий делитель (НОД)** — найдите НОД двух чисел.
* **Арифметическая прогрессия** — найдите пропущенное число в последовательности.
* **Простое ли число?** — ответьте «yes», если число простое, и «no» в противном случае.

Вы победите, если дадите три правильных ответа.




## Установка и запуск

**1. Установка зависимостей:**
make install

**2. Сборка пакета:**
make build

**3. Установка собранного пакета:**
make package-install

**4. Запуск игр:**
brain-even — проверка на чётность;
brain-calc — калькулятор;
brain-gcd — наибольший общий делитель;
brain-progression — арифметическая прогрессия;
brain-prime — простое ли число?
